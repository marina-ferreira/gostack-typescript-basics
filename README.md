# GoStack 12
## Week 1 - TypeScript Fundamentals

**TypeScript Express Projects API**

Create, show, list, edit and delete projects.

```
GET /repositories
GET /repositories/:id
POST /repositories
PUT /repositories/:repository_id
DELETE /repositories/:repository_id
```

Dependencies:

* Node - v12.17.0
* Express - v4.17.1
* TypeScript - v3.9.5
